# Метаанализ диагностической точности моделей ИИ в цитологии шейки матки

## ОПИСАНИЕ ПРОЕКТА
Данный репозиторий содержит код и данные для воспроизведения результатов систематического обзора и метаанализа диагностической точности моделей искусственного интеллекта (ИИ) для классификации цитологических изображений шейки матки.

## ОСНОВНЫЕ РЕЗУЛЬТАТЫ
- **Количество включенных исследований:** 16
- **Чувствительность:** 95.6% (95% ДИ: 92.7–97.4%)
- **Специфичность:** 95.3% (95% ДИ: 91.4–97.4%)
- **AUC (площадь под SROC-кривой):** 0.981 (95% ДИ: 0.956–0.994)
- **LR+:** 20.18 (значительное увеличение вероятности)
- **LR-:** 0.05 (значительное уменьшение вероятности)
- **Гетерогенность:** I² > 95% (значительная)
- **Публикационный сдвиг:** выявлен (тест Дикса, p = 0.011)
