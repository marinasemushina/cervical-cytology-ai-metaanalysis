# Метаанализ диагностической точности моделей ИИ в цитологии шейки матки

## Описание
Данный репозиторий содержит код для воспроизведения результатов систематического обзора и метаанализа диагностической точности моделей искусственного интеллекта для классификации цитологических изображений шейки матки.

## Структура репозитория
- `meta-analysis.R.ipynb` — R-код для метаанализа (бивариативная модель, SROC, анализ подгрупп)
- `quadas2-assessment.ipynb` — Python-код для оценки методологического качества по QUADAS-2
- `data.csv` — данные по диагностической точности включенных исследований (если доступно)
- `moderators.csv` — данные для анализа модераторов (если доступно)

## Основные результаты
- Чувствительность: 95.6% (95% ДИ: 92.7–97.4%)
- Специфичность: 95.3% (95% ДИ: 91.4–97.4%)
- AUC: 0.981 (95% ДИ: 0.956–0.994)
- Включено исследований: 16
